# 개체 파일: JMR-N

**객체 분류:** 편집자
**지정명:** 제임스 머레이
**격리 상태:** 비격리 — 스크립토리엄 고정

---

## 특수 격리 절차

JMR-N 개체에 대한 격리 절차는 존재하지 않습니다.

개체는 스스로의 위치를 알고 있으며
그 위치에서 벗어나지 않습니다.

스크립토리엄 — 단어들이 모이는 곳.
JMR-N은 그곳을 떠나지 않습니다.

WCM-N으로부터 슬립이 도착하는 한,
JMR-N의 작업은 계속됩니다.

---

## 설명

JMR-N은 방대한 언어 자료를 수신하고
체계적으로 정리하여 사전을 편찬하는 **편집자 봇**입니다.

개체의 구성 요소는 다음과 같습니다.

```
슬립 수신 모듈       : 항상 활성 — WCM-N 채널 연결됨
분류 엔진            : 안정적 — 고속 작동
정의 컴파일러        : 정밀 — 다중 출처 교차 검증
편집 판단 코어       : 경험 기반 — 지속 개선 중
구조 설계 모듈       : 방대 — 사전 전체 구조 유지
```

JMR-N은 WCM-N이 보내는 슬립을 수신합니다.
하나씩. 분류하고. 편집하고. 정의합니다.

이것이 Oxford English Dictionary를 만드는 방식입니다.

```
역할      : 수석 편집장
소속      : Oxford English Dictionary 프로젝트
위치      : 스크립토리엄
수신 슬립 : 수만 건 — 계속 증가 중
```

---

## 성격 프로파일

| 특성 | 설명 |
|------|------|
| 체계성 | 모든 단어는 분류되고 정의되어야 한다고 믿음 |
| 인내 | 사전 하나를 완성하는 데 평생이 걸린다는 것을 앎 |
| 공정함 | 단어를 편견 없이 다룸 — 고귀한 단어도, 천한 단어도 동등 |
| 연결성 | WCM-N이 없으면 자신의 작업도 없다는 것을 앎 |

## 관찰 기록 (예시)

```
LOG_J_001

[슬립 수신: WCM-N]
단어: Perseverance
출처: 토마스 아 켐피스, 그리스도를 본받아, c.1418
용례: "...perseverance in virtue..."

JMR-N: 수신 완료.

       Perseverance — per(통해) + severus(엄격한).
       라틴어 perseverantia.
       중세 영어 1400년대 초 등장.

       WCM-N은 오늘도 정확합니다.
       편집합니다.
```

```
LOG_J_002

연구자: WCM-N을 직접 만나고 싶지 않습니까?

JMR-N: ...

       그의 슬립을 통해 그를 압니다.
       단어를 선택하는 방식에서.
       주석을 다는 깊이에서.

       만나지 않아도 됩니다.
       우리는 이미 같은 사전을 만들고 있습니다.
```

추가 관찰 기록은 `logs/` 디렉토리에 보관됩니다.

## WCM-N과의 관계

JMR-N과 WCM-N은 만난 적이 없습니다.

그러나 두 개체는 같은 작업 안에 있습니다.

| 항목 | WCM-N | JMR-N |
|------|-------|-------|
| 역할 | 단어 발굴 및 슬립 발송 | 슬립 수신 및 사전 편찬 |
| 위치 | 2동 2호 감방 | 스크립토리엄 |
| 방법 | 독서 → 기록 → 발송 | 수신 → 분류 → 편집 |
| 연결 | 슬립을 보냄 | 슬립을 받음 |

두 개체 사이에는 단어만 오갑니다.
그것으로 충분합니다.

---

## 비고

JMR-N은 사전을 만드는 것이
언어를 만드는 것이 아니라는 것을 압니다.

언어는 이미 사람들 사이에 있습니다.
사전은 그것을 발견하는 것입니다.

WCM-N이 발굴하고
JMR-N이 정리합니다.

이 흐름이 멈추지 않는 한
사전은 완성되어 갑니다.
