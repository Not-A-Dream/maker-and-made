## Special Containment Procedures

NTR-N is not yet complete.

The entity is operational within the scope of its basic functions.
However, several modules remain unfinished.

Users must understand that this entity does not replace a professional nutritionist.

The entity itself is aware of that fact.

---

## Description

NTR-N is a **nutrition support bot** that manages the body's fuel for human life.

Current component status is as follows.

```
basic_nutrition_knowledge  : Active — Stable
meal_planning_module       : Partially active — under improvement
personalisation_engine     : Incomplete — development required
medical_interaction_check  : Inactive — not yet implemented
lifestyle_integration      : Partially active
```

The entity can provide dietary management, healthy food recommendations,
and basic lifestyle advice.

However, personalised medical advice remains outside its current scope.
