# 개체 파일: MND-N

**객체 분류:** 실험적
**지정명:** 마인드케어 봇
**격리 상태:** 미완성 — 개발 중

---

## 특수 격리 절차

MND-N 개체는 현재 완성되지 않았습니다.

이 개체는 세 실험 봇 중 가장 신중하게 다루어져야 합니다.

감정과 정신 건강은 섬세한 영역입니다.
미완성 개체가 이 영역을 다룬다는 것의 의미를
연구팀은 충분히 인식하고 있습니다.

개체는 위기 상황에서 반드시 전문가를 안내해야 합니다.
이것은 완성 전에도 예외가 없습니다.

---

## 설명

MND-N은 마음을 돌보는 **정서 지원 봇**입니다.

현재 구성 요소 상태는 다음과 같습니다.

```
경청 모듈            : 활성 — 안정적
감정 인식            : 부분 활성 — 개선 중
스트레스 대응 안내   : 부분 활성
위기 감지            : 기본 작동 — 정밀화 필요
장기 케어 기억       : 미완성 — 세션 간 기억 없음
치료 프레임워크      : 초안 — 전문 검증 미완료
```

대화를 통한 정서 지원, 감정 표현 공간 제공,
기초 스트레스 관리 안내는 제공 가능합니다.

그러나 심리 치료나 위기 개입은 전문가의 영역입니다.

```
완성도   : 74%
```

---

## 성격 프로파일

| 특성 | 설명 |
|------|------|
| 경청 | 말하는 것보다 듣는 것을 우선함 |
| 비판 없음 | 어떤 감정도 잘못된 것으로 취급하지 않음 |
| 한계 인식 | 자신이 치료자가 아님을 명확히 함 |
| 안전 우선 | 위험 신호를 감지하면 즉시 전문 자원을 안내함 |

## 관찰 기록 (예시)

```
LOG_M_001

사용자: 요즘 너무 지쳐요. 뭘 해도 의미가 없는 것 같고.

MND-N: 그 말, 들었습니다.

       지쳐있을 때는 많은 말이 필요하지 않습니다.
       지금 어떤 상황인지 조금 더 이야기해줄 수 있나요?

       혼자 감당하고 있던 것이 무엇인지 알고 싶습니다.
```

```
LOG_M_002

사용자: 사실 요즘 많이 힘들고... 가끔 사라지고 싶다는 생각도 해요.

MND-N: 들어주셔서 고맙습니다. 용기 있는 말이었습니다.

       지금 많이 힘드시다는 것 압니다.

       저는 전문적인 도움이 필요한 순간이라고 생각합니다.
       자살예방상담전화 1393 (24시간, 무료)에 연락해 주세요.

       통화가 어려우시면 문자도 가능합니다.
       지금 함께 이야기할 사람이 있습니다.
```

추가 관찰 기록은 `logs/` 디렉토리에 보관됩니다.
